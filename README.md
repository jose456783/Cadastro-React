# 📋 Cadastro React

Um projeto acadêmico de aplicação web moderna desenvolvido com **React** e **Vite**, demonstrando conhecimentos em componentes React, gerenciamento de estado e UI responsiva com Bootstrap.

---

## 🎯 Objetivo

Criar uma interface intuitiva e funcional para cadastro de usuários, aplicando boas práticas de desenvolvimento front-end com React e ferramentas modernas.

---

## 🛠️ Tecnologias Utilizadas

- **React 18.2** - Biblioteca para construir interfaces
- **Vite 4.4** - Build tool ultra-rápido
- **Bootstrap 5.3** - Framework CSS para estilização
- **React-Bootstrap 2.10** - Componentes Bootstrap em React
- **JavaScript ES6+** - Linguagem base

---

## 📦 Funcionalidades

✅ **Formulário de Cadastro** - Campos para nome, email e senha  
✅ **Validação de Entrada** - Captura de dados com React Hooks  
✅ **Modal de Confirmação** - Feedback visual ao usuário  
✅ **Navegação** - Header com menu responsivo  
✅ **Design Responsivo** - Interface adaptável a diferentes telas  

---

## 🚀 Como Executar

### Pré-requisitos
- Node.js (v14+)
- npm ou yarn

### Instalação

```bash
# Clonar repositório
git clone https://github.com/jose456783/Cadastro-React.git
cd Cadastro-React

# Navegar para a pasta do projeto
cd projeto-vite2

# Instalar dependências
npm install
```

### Executar em Desenvolvimento

```bash
npm run dev
```

A aplicação estará disponível em `http://localhost:5173`

### Build para Produção

```bash
npm run build
```
